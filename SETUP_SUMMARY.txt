⚡ HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════

LOCAL FLOW:
┌──────────┐         ┌─────────────┐         ┌──────────┐
│ Browser1 │────────>│ ws://        │<────────│ Browser2 │
│ Player 1 │ WebSock │ localhost:   │ WebSock │ Player 2 │
│  (Host)  │<────────│  8081       │────────>│ (Joiner) │
└──────────┘         └─────────────┘         └──────────┘
     ↑                      ↑                      ↑
     └─────────────────────┬──────────────────────┘
         python api/index.py (on port 8081)


NETLIFY FLOW:
┌──────────┐         ┌────────────────────────┐         ┌──────────┐
│ Browser1 │────────>│ wss://                 │<────────│ Browser2 │
│ Player 1 │ WebSock │ yoursite.netlify.app   │ WebSock │ Player 2 │
│  (Host)  │<────────│ /.netlify/functions/   │────────>│ (Joiner) │
│          │         │   index/ws             │         │          │
└──────────┘         └────────────────────────┘         └──────────┘
     ↑                      ↑                      ↑
     └─────────────────────┬──────────────────────┘
   Netlify CDN + Serverless Function

🆘 COMMON ISSUES & FIXES
═══════════════════════════════════════════════════════════════════════════

"WebSocket connection failed"
└─ CHECK: Is server running? python api/index.py
└─ FIX: Start the server in a terminal first

"Game loads but won't connect"
└─ CHECK: Browser console for error messages
└─ CHECK: Is localhost:8081 reachable? (check firewall)
└─ FIX: Refresh page, restart server

"Second player won't join"
└─ CHECK: Are both browsers on same URL?
└─ CHECK: Did first player click "Online"?
└─ FIX: Refresh second browser, try again

"Netlify deployment failed"
└─ CHECK: Do api/ and index.py exist?
└─ CHECK: Is netlify.toml in root?
└─ FIX: Check Netlify dashboard build logs
